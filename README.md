# ESP32 Multi-Sensor Development Board

## Overview
Designed and developed a custom ESP32-based embedded system integrating multiple sensors for real-time monitoring and visualization.

## Hardware Components
- ESP32 Microcontroller
- NEO-6M GPS Module
- MQ2 Gas Sensor
- AHT10 Temperature & Humidity Sensor
- OLED Display

## Features
- Real-time sensor data acquisition
- Gas detection and environmental monitoring
- GPS-based location tracking
- On-device display via OLED

## Technologies Used
- C/C++
- Embedded Linux / Arduino
- I2C, UART communication
- PCB Design (EasyEDA)
